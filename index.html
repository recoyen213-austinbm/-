```

世界變了。
我們的孩子，準備好了嗎？

2030年，今天入學的孩子將要面對的工作，有65%現在還不存在。這不是危言聳聽，這是牛津大學的研究數據。AI 會處理訊息，機器會執行指令，但有一件事，只有人類能做——

「在充滿不確定的世界裡，
看見可能性、連結他人、創造意義。」

探索者學院從這個問題出發：如果我們只有八週，我們能給孩子什麼，是他們帶得走、用一輩子的？

不是單字量。不是文法分數。不是一張証書。
而是一種思維方式、一種與人相處的能力、還有一個深深刻在心裡的信念—— 「我可以面對任何挑戰。」

```
為什麼我們不一樣

大多數暑期課程教孩子知道什麼。
我們教孩子成為什麼。

```
01

英語是載具，不是目的地

我們用雙語環境包裹每一個挑戰——解謎、談判、協作、表達。孩子在真實情境中使用英語，不是為了考試，而是因為有話要說、有事要做。語言能力在不知不覺中，成為了本能。

02

每堂課都有一個「峰值時刻」

諾貝爾獎心理學家告訴我們：人記住的不是平均，而是高峰與結尾。我們刻意設計每一堂課的高峰體驗。而第八週的最後一堂——燈光儀式入場、傳奇石宣誓、老師唸出每人專屬稱號、時光膠囊——是整個夏天最高的那個峰值，孩子帶著「我真的做到了」的感覺離開。

03

七個習慣，不是口號

我們把《與成功有約》的七個習慣，拆解成孩子能體驗的遊戲、角色扮演、真實衝突的解決。不是上課學，是活出來。孩子離開時帶走的，是一套面對世界的工具。

探索者學院三大核心

八週，三條線，交織成一個完整的孩子。

雙語力

BILINGUAL POWER

不只是會說英語，而是能用英語思考、感受、說服、連結。我們用沉浸式情境，讓英語成為孩子表達自己的自然語言。

🧭

內在羅盤

INNER COMPASS

七個習慣是每個探索者的導航系統。主動積極、以終為始、雙贏思維——這些不是大道理，是每一天可以選擇的行動。

🎨

創造者精神

MAKER MINDSET

才藝不是表演，是思維。無論是偵探的推理、科學家的實驗、藝術家的創作，我們培養的是「看見問題、動手解決」的基因。

```
一封信
寫給每一位考慮報名的家長

親愛的爸爸媽媽，

我理解你的猶豫。暑假很寶貴，孩子的時間更寶貴。你不想讓他去一個地方只是坐著、聽著、然後忘掉。

我也是這樣想的。所以我們花了很長時間，重新問自己這個問題：八週能讓一個孩子發生什麼真正的改變？

我們找到的答案，不是更多的單字、更難的文法。而是讓孩子真實地體驗——在壓力下做決定、在衝突中找雙贏、在陌生語言裡表達自己的想法、在最後一天對著全班大聲說出：「我做到了我以為自己做不到的事。」

這個瞬間，才是我們最想給你孩子的東西。

我們叫它探索者學院，因為每個孩子都是探索者。他們探索語言、探索習慣、探索自己的邊界。而我們，只是那個相信他們、並且陪他們走過這段旅程的人。
